# Elvi Project Skeleton

1. Clone this repository
2. `composer install`
3. `bin/robo skeleton:init <git-remote-url>`
4. Review and add missing parameter values in `.env` and `.env.(elvidev|elvitest|elviapp)` files
5. Have a look around with some demo data. When done, remove demo data with `bin/robo demo:remove`
6. Add relevant data to `README.md` file
7. Commit and push everything to the new repository
