# ELVI Consent 

This repository contains the Elvi Consent context, a Symfony-based application that manages customer consent for data processing and sharing.

## Requirements

- PHP 8.4 or higher
- Composer
- Docker

## Installation

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/vision-group/elvi-consent.git
    cd elvi-consent
    ```

2.  **Install dependencies:**

    ```bash
    composer install
    ```

3.  **Configure environment variables:**

    Create a `.env.local` file by copying the `.env` file:

    ```bash
    cp .env .env.local
    ```
